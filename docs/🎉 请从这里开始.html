🧱 内容块

在思源笔记中，唯一重要的核心概念是内容块。我们可以使用自然的 Markdown 语法来定义内容块，在内容块的粒度上对思绪和知识进行整理，同时也能方便长文排版输出。

🔮 快速上手

  1. 打开本地文件夹或者连接 WebDAV
  2. 通过输入 (( 来触发内容块引用
  3. 在内容块中遨游
  4. 窗口和页签
