Vditor 编辑器

思源笔记的编辑器组件叫做 “Vditor”，它是一款浏览器端的 Markdown 编辑器，也是由我们设计和开发的开源项目，可以在这里找到它的代码。我们基于 Vditor 开发了现在思源笔记中使用的编辑器模式，未来将继续进行完善，给大家带来更现代化的 Markdown 编辑体验。
